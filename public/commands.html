Commands

Initialization

$ picogen init

Generate

$ picogen generate

Generates static files.

Arguments Descriptoin type
-s Create a sitemap.xml along with static files Bool
-f Generate feed.xml for posts Bool
-p to specify port. Number
-P to run in production mode. Bool

Server

$ picogen server

Starts the server.

Arguments Descriptoin type
-n start server with no auto reload. Bool
-i to specify wait time for auto reload( in ms ). Number
-p to specify port. Number
-P to run in production mode. Bool

Create

$ picogen create <json file> <output path>

create pages.

Clean

$ picogen clean

Cleans the cache file (db.json) and generated files (public).
