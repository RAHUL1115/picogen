Configuration

settings description type

title feed title String
description feed description String
copyright feed copyright String
image feed image String
favicon feed favicon String
removehtmlext Whether to remove the .html extension or not from server links. String
removehtmlextgen Whether to remove the .html extension or not from generated files. String
